# 🎓 ЕЦМП ЧР — автоматизированная система передачи данных в ФЦМПО

## 📖 Описание проекта
ВИДЕО-ОПИСАНИЕ ПРОЕКТА И ИНСТРУКЦИЯ: https://youtu.be/gEoG6vspkOc?si=NbmBMl6HeIefbAwz

**ЕЦМП** — это современная система для автоматизации обмена данными между образовательными учреждениями и  
**Федеральным центром мониторинга обучающихся (ФЦМПО)**.  

Сервис предназначен для сбора, проверки и распределения отчётных данных по питанию обучающихся, а также для централизованного управления учреждениями на уровне районов и регионов.  
Он обеспечивает безопасную загрузку и публикацию актуальных отчётных файлов меню, доступных для автоматизированных систем ФЦМПО 24/7.

---

## 🎯 Основные цели

- Повышение прозрачности и скорости обмена данными между учреждениями и ФЦМПО.  
- Исключение ручной работы и человеческого фактора при передаче отчётов меню.  
- Централизованное администрирование образовательных организаций.  
- Возможность массовой рассылки, обновления и синхронизации данных “в одно действие”.

## ⚙️ Технологический стек

| Компонент | Технология |
|------------|-------------|
| **Ядро приложения** | Python3.11 + FastAPI |
| **Веб‑сервер / балансировка** | Nginx |

## 🧩 Функционал системы

### 👩‍💼 Разделение ролей и уровней доступа

#### **1. Районный администратор**
- Управляет учреждениями внутри своего района.  
- Может отправлять отчёты,приказы,письма или инструкции всем подведомственным учреждениям одной кнопкой.  
- Имеет возможность загружать обновлённые файлы меню (`.xlsx`) централизованно.  
- Получает информацию о статусе приёма файлов каждым учреждением.  

#### **2. Региональный администратор**
- Имеет полный контроль над всеми районами региона.  
- Может выполнять рассылку отчётов и уведомлений сразу по всем районам или выборочным группам.  
- Получает агрегированные данные по всем учреждениям региона в реальном времени.  
- Может централизованно “взять под управление” любое учреждение (в режиме просмотра или редактирования) **одним нажатием**.  
- Имеет возможность перейти напрямую в **веб‑версию 1С**, если у него есть соответствующие права доступа (для сверки отчётных данных, актуализации питания и автозагрузки сводок).

---

### 🗂 Управление учреждениями

- Каждое учреждение закреплено за конкретным районом.  
- Структура данных повторяет административное деление:  
  ```
  /region/
      /district_1/
          /school_1/
              findex.xlsx
          /school_2/
              findex.xlsx
      /district_2/
          /school_3/
              findex.xlsx
  ```
- Администраторы видят статус обновления по каждому учреждению, дату последней загрузки и формат отчёта.

---

### 💌 Рассылка и обновление файлов

- Районные и региональные администраторы могут отправить файлы xlsx (`findex.xlsx и другие`) сразу на все учреждения по типу питания.  
- Система автоматически распределяет его по соответствующим каталогам.  
- Есть просмотр истории рассылок и уведомления об успешной загрузке.  
- При необходимости можно инициировать повторную рассылку “в одно нажатие”.

---

### 🖥 Интеграция с 1С

- Реализована ссылка‑переход в **веб‑версию 1С** (при наличии доступа у ответственного лица).  
- Нужна для сверки данных о питании, меню и количестве обучающихся.  
- Переход безопасен: используется авторизованный аккаунт администратора региона.  

---

### 🔗 Взаимодействие с ФЦМПО

Система формирует и публикует файлы, доступные по постоянным ссылкам (например):
https://monitoring95.ru/2/food/findex.xlsx

ФЦМПО‑роботы автоматически обращаются к этим адресам, загружая свежие данные по расписанию.
Все файлы передаются в требуемых форматах и находятся на российских серверах, что соответствует техническим требованиям центра.

## 🧠 Преимущества решения

- Полная автоматизация обмена данными между учреждениями, районными администраторами и ФЦМПО.  
- Централизованное управление всеми объектами из единой панели.  
- Поддержка гибкой иерархии пользователей (регион/район/учреждение).  
- Доступ к актуальной информации для всех уровней управления образования.   
- Поддержка интеграции с системами 1С для унификации учётных данных.
- Возможность управлять всеми учреждениями своего района для районных админов
- Возможность управлять всеми учреждениями региона для регионального админа
- Авторасссылка файлов ЕМ и ТМ для учреждений региона/района
- Справочник документов, приказов, инструкций от регионального админа и возможность скачивать эти документы
- Возможность делать поиск в справочнике документов

## 🔒 Безопасность

- Все соединения — через HTTPS 
- Маршруты и IP‑адреса проходят только внутри Российской Федерации (хостинг Timeweb Cloud или reg.ru).  
- Уровни доступа разграничены по ролям и авторизации пользователей.  
- Логи автоматически очищаются и архивируются.
